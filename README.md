## Laravel Jetstream Inertia Vue PrimeVue Starter

![Screenshot](docs/screen.png)


Starter kit to accelerate the development of new applications.

### Features
* Laravel 8
* Inertia.js
* Jetstream
* Vue 3
* PrimeVue
* Sakai PrimeVue Theme
* Vue-i18n
* Spatie Permissions
* Spatie Settings
* Spatie View Models
* Spatie Data Transfer Objects
